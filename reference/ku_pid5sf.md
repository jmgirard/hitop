# Real PID-5-SF Data

Real responses to items on the PID-5-SF (with 100 items) from University
of Kansas students.

## Usage

``` r
ku_pid5sf
```

## Format

### `ku_pid5sf` A [tibble](https://tibble.tidyverse.org/reference/tibble.html) with 386 rows and 101 columns.

- response_id:

  An anonymized id for each participant

- pid_1 to pid_100:

  Responses on each item
