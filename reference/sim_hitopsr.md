# Simulated HiTOP-SR Data

Simulated responses to items on the full HiTOP-SR (with 405 items).

## Usage

``` r
sim_hitopsr
```

## Format

A [tibble](https://tibble.tidyverse.org/reference/tibble.html) with 100
rows and 405 columns.

- hsr_1 to hsr_405:

  Responses on each item
